# Bakabase

**Local Media Manager** - Keep your media library organized

Bakabase is an offline media management software designed for managing local media files. It supports fast processing, smart searching, convenient previewing, and playback.

## Features

- **Multi-type Support** - Anime, comics, audio dramas, doujinshi, movies, image collections, and more
- **Smart Organization** - Automatic file structure recognition with a flexible property system
- **Metadata Enhancement** - Auto-fetch covers, tags, and metadata from multiple sources
- **Efficient Search** - Property-based fast filtering and searching
- **Built-in Player** - Direct preview for images, videos, and other media
- **Download Tools** - Built-in downloaders for Bilibili, ExHentai, Pixiv, and more

## Download

- [Stable](https://github.com/anobaka/Bakabase/releases/latest) - Reliable, recommended for daily use
- [Beta](https://github.com/anobaka/Bakabase/releases) - Latest features, may contain unknown issues
