# Bakabase

**本地媒体管理器** - 让你的本地媒体库井井有条

Bakabase 是一款离线媒体管理软件，专为管理本地媒体文件设计。支持快速处理、智能搜索、便捷预览与播放。

## 特性

- **多类型支持** - 动画、漫画、音声、本子、电影、图集等
- **智能分类** - 自动识别文件结构，灵活的属性系统
- **元数据增强** - 支持多种数据源自动获取封面、标签等信息
- **高效搜索** - 基于属性的快速筛选与搜索
- **内置播放器** - 支持图片、视频等媒体的直接预览
- **下载工具** - 内置 Bilibili、ExHentai、Pixiv 等下载器

## 下载
